# Auryk Holding Group

Sitio corporativo estático con **Auryk Control**, un CMS administrativo en HTML, CSS y JavaScript puro conectado a Supabase mediante Fetch API.

- Sitio público: `/index.html`
- Administración: `/admin/`
- Esquema: `/supabase/setup.sql`
- Configuración: copia `config.example.js` como `config.js`
- Documentación: `/docs/`

La integración mantiene el contenido original como fallback si Supabase no está configurado o no responde.
